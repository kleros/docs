# V1 Protocol Documentation

This section documents the Kleros V1 protocol and its products, along with retired applications. V1 is the production-proven protocol on Ethereum L1 and remains active for web3 integrations.

<Note>
Each V1 product below has a V2 version available on Arbitrum.
</Note>

## What V2 Changes

- Deployed on Arbitrum instead of Ethereum L1
- Modular dispute kit architecture
- Cross-chain support via VEA bridge
- Evidence and appeals handled by the Court rather than the arbitrable contract
- Vote privacy through Shutter integration (in development)
