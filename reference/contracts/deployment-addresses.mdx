## Finding Current Addresses

The most reliable source is the repository:
```bash
git clone https://github.com/kleros/kleros-v2.git
cd kleros-v2/contracts/deployments
cat arbitrum/KlerosCore.json | jq '.address'
```

All V2 core contracts use UUPS proxy pattern. Proxy addresses remain stable across upgrades.
