## Integration Categories

### Kleros-Built Products

Kleros Court powers all of these directly:

| Product | Description | Link |
| --- | --- | --- |
| **Court** | Core dispute resolution protocol | [court.kleros.io](https://court.kleros.io) |
| **Curate** | Decentralized lists and registries | [curate.kleros.io](https://curate.kleros.io) |
| **Escrow** | Secure transactions with built-in arbitration | [escrow.kleros.io](https://escrow.kleros.io) |
| **Scout** | Contract and token verification | [scout.kleros.io](https://scout.kleros.io) |
| **Proof of Humanity** | Sybil-resistant identity registry | [proofofhumanity.id](https://proofofhumanity.id) |
| **Governor** | DAO governance execution | [governor.kleros.io](https://governor.kleros.io) |
