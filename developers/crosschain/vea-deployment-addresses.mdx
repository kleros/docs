## Mainnet Deployments

For production mainnet contract addresses across all supported routes (Arbitrum to Ethereum, Arbitrum to Gnosis), refer to the canonical source:

- [Vea Deployment Addresses on docs.vea.ninja](https://docs.vea.ninja/introduction/deployment-addresses)
- [Vea GitHub repository deployment artifacts](https://github.com/kleros/vea)

<Note>
Mainnet addresses should be pulled from the Vea repository's deployment artifacts. This page will be updated as mainnet deployments are finalized and verified.
</Note>

## How to Find Addresses

Clone the Vea repository to inspect deployment artifacts:

```bash
git clone https://github.com/kleros/vea.git
cd vea
# Check contracts/deployments/ for per-chain deployment JSONs
```
